Create Post


콘텐츠 작성·게시

‘Create Post’ 태스크는 사용자가 콘텐츠를 작성하여 게시할 수 있는 모든 서비스 카테고리에 나타나는 특징이다. 일반적으로 서비스에서 노출하고자 하는 콘텐츠('Content', 'News Feed' 태스크) 규격이 설계된 후, 사용자가 규격화된 정보를 작성할 수 있도록 입력 필드를 제공한다.

‘Create Post’ 태스크를 제공하는 서비스는 사용자가 작성한 콘텐츠의 질과 양이 서비스의 핵심 성장 지표가 되는 경우가 많으므로 사용자가 규격 내에서 빠르게 콘텐츠를 작성할 수 있도록 설계한다. 규격화된 정보에 대한 입력 필드를 작성할 경우 추천값을 노출하여 콘텐츠의 질을 유지하면서 빠르게 입력할 수 있도록 하며, 입력 필드가 많거나 콘텐츠 작성을 위해 많은 시간을 필요로 하는 경우 한 페이지가 아닌 여러 페이지의 단계형으로 분산하여 사용자가 과업 작성을 포기하지 않고 완수할 수 있도록 독려한다.

‘Create Post’ 태스크 화면 설계
한 페이지 형 (One page)

사용자가 즉각적으로 규격화된 정보를 작성할 수 있는 생산성 서비스나 소셜 네트워킹 서비스의 경우 한 페이지에 입력 필드를 노출한다.

단계형 (Steps)

사용자가 규격화된 정보 작성을 위해 많은 시간을 소모해야 하는 여행, 식음료 등 큐레이션 서비스의 경우 일반적으로 한 페이지에서 한 가지 종류의 정보를 입력할 수 있도록 설계한다. 각 페이지에 전체 단계 중 현재 단계를 파악할 수 있는 프로그레스 바(progress bar)를 노출하여 사용자가 과업을 완수할 수 있도록 독려한다.
